# Multi-client measurements

This folder includes two scripts:
- measure
- plot

## Usage:

1. Install requirements with pip (virtualenv or conda recommended)
2. Create several Unity builds with different audio settings (e.g., number of audio sources)
2. Change the settings in the measure-script
   - Most importantly, the path to the Unity build
1. Run measurements using the measure-script
2. Plot results using the plot-script
